  6. HOW TO VIEW THE WEBSITE
================================================================================

  OPTION 1: Direct Open (Simplest)
  ──────────────────────────────────────────────────────────────────────
  1. Navigate to the project folder: MP-Projects-HubJS/
  2. Double-click on "index.html"
  3. The homepage will open in your default browser

  NOTE: Some features (ES modules, absolute paths) work best with
  a local server. If navigation links don't work from file://,
  use Option 2.

  OPTION 2: Local Server (Recommended)
  ──────────────────────────────────────────────────────────────────────
  If you have Node.js installed:
    npx serve .

  If you have Python installed:
    python -m http.server 8080

  Then open: http://localhost:8080

  OPTION 3: VS Code Live Server
  ──────────────────────────────────────────────────────────────────────
  1. Open the project folder in VS Code
  2. Install the "Live Server" extension
  3. Right-click index.html → "Open with Live Server"

  NAVIGATION FLOW
  ──────────────────────────────────────────────────────────────────────
  Homepage (index.html)

    │
    └── Footer shows "Projects Hub" brand + developer credit

  From any inner page you can navigate to:
    - Home page

  ──────────────────────────────────────────────────────────────────────

  opening index.html directly via double-click (file:// protocol),
  the homepage will work but inner page icons may not display.
  Solution: Use a local server as described in Section 6.

  The CSS file for the thank-you page is named "תדה.css" while the
  HTML page is "תודה.HTML" (different spelling). This is a pre-existing
